# mypkg

* 本リポジトリは，ROS2のパッケージです．Talkerを起動させてから経過した時間をListenerで受信し，表示するものです．
* 本リポジトリを使用するにはROS2をあらかじめインストールしている必要があります．
* 本リポジトリは，2023年度の千葉工業大学先進工学部未来ロボティクス学科のロボットシステム学の講義で取り扱われた練習用のリポジトリです．
